# @sys/std
Standard libraries

Surfaces, and lightly augments when necessary, the deno "[@std libs](https://jsr.io/@std)".

### System Namespace:

- [`jsr: @sys/std`](https://jsr.io/@sys/std)
- [`jsr: @sys/std-server`](https://jsr.io/@sys/std-server)


#### Example

```ts
import { Path } from '@sys/std';
```
