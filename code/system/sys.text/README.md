## Markdown


Grammars:
- [CommonMark](https://commonmark.org/)
- GFM ([Github Flavored Markdown](https://github.github.com/gfm/))


### GFM (Github Flavored Markdown)

```
# GFM

## Autolink literals

www.example.com, https://example.com, and contact@example.com.

## Footnote

A note[^1]

[^1]: Big note.

## Strikethrough

~one~ or ~~two~~ tildes.

## Table

| a | b  |  c |  d  |
| - | :- | -: | :-: |

## Tasklist

* [ ] to do
* [x] done
```
